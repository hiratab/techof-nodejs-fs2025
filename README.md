# What is Node.js
Node.js is runtime environment that lets you execute Javascript on the server. It uses V8 engine and provides an event-driven, non-blocking Input/Output (I/O) model that makes it lightweight and efficient.
- Javascript outside of the browser
- Google's V8 engine

tl/dr => Javascript + Node.js => Backend code that runs on a server


## What we can be built using Node.js
- Web servers & API
- Real-time apps => Chat apps, collaboration tools, live dashboards
- Backend for mobile/website apps 
- Microservices => independent services
- Command-line tools
- Automation => File processing, cron job, data pipelines
- Integration => APIs, webhooks, bots
